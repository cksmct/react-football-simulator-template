# React Football Simulator Template ⚽

A lightweight, client-side React boilerplate for building hardcore football/soccer draft simulators. It includes basic RNG (Random Number Generator) match simulation logic and a clean state management structure.

## 🚀 Live Production Example

Want to see what you can build with this logic? 

Check out our full production build here: **[8-0 World Cup Draft Simulator](https://8-0.online)**. 
We expanded this basic template into a brutally hardcore simulator featuring real FC 24 stats, multi-language support, and a cinematic dark mode.

## 📦 Features
- No backend required (pure client-side React logic).
- Basic weighted RNG for match simulation based on team ratings.
- Easy to extend for different tournaments.

## 🎮 Game Modes You Can Build
Using this template, you can easily replicate complex game flows. For instance, in our live deployment, we built the **[2026 Random XI Challenge](https://8-0.online/)** mode, which forces players to win 8 consecutive matches with entirely randomized underdog squads.

## 🛠️ Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Modify `src/simulateMatch.js` to adjust the RNG harshness.
